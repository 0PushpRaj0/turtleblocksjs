1. Welcome to Turtle Blocks

    Turtle Blocks is a Logo-inspired turtle that draws colorful pictures with snap-together visual-programming blocks.

  2. Palette buttons

    This toolbar contains the palette buttons: click to show or hide the palettes of blocks (Turtle, Pen, Numbers, Boolean, Flow, Blocks, Media, Sensors, and Extras).

  3. Block palettes

    This is the Turtle Palette, which is shown when you click on the Turtle Button. Drag these blocks onto the canvas to use them. To open other palettes, click on the palette buttons on the far left.

  4. Expand/collapse toolbar

    This button opens and closes the primary toolbar.

  5. Run fast

    Click to run the project in fast mode.

  6. Run slow

    Click to run the project in slow mode.

  7. Stop

    Stop the current project.

  8. Clean

    Clear the screen and return the turtles to their initial positions.

  9. Show/hide palettes

    Hide or show the block palettes.

  10. Show/hide blocks

    Hide or show the blocks and the palettes.

  11. Expand/collapse collapsable blocks

    Expand or collapse stacks of blocks, e.g, start and action stacks.

  12. Expand/collapse option toolbar

    Click this button to expand or collapse the auxillary toolbar.

  13. Paste

    The paste button is enabled then there are blocks copied onto the clipboard.

  14. Cartesian

    Show or hide a Cartesian-coordinate grid.

  15. Polar

    Show or hide a polar-coordinate grid.

  16. Load samples from server

    This button open a viewer for loading example projects.

  17. Load project from file

    You can also load projects from the file system.

  18. Delete all

    Remove all content on the canvas, including the blocks.

  19. Undo

    Restore blocks from the trash.

  20. Save project

    Save your project to a server.

  21. Welcome

    Congratulations, you have finished the tour. Please enjoy Turtle Blocks!
